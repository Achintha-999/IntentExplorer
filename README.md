# Leaderboard Project - C++

## Project Details

## Requirements
Ensure your environment has the following installed:
- C++17 or a higher compiler.
- CMake 3.10 or higher.
- A supported operating system:
  - Windows
  - Linux
  - macOS
- Git (for cloning the repository).

## How to Build and Run
Follow the steps below to build and execute the project:

1. Clone the repository:
```bash
git clone https://github.com/Achintha-999/leaderboard-project-cpp.git
```

2. Navigate into the project directory:
```bash
cd leaderboard-project-cpp
```

3. Build the project using CMake:
```bash
mkdir build
cd build
cmake ..
make
```

4. Run the executable:
```bash
./leaderboard_project
```

## Usage
1. Follow the prompts in the terminal to interact with the leaderboard system.
2. Add, view, and update player details.
3. Utilize sorting functionality to structure the leaderboard as needed.
